הוראות ליצירת תמונת Open Graph

  1. לחץ F12 לפתיחת Developer Tools
  2. לחץ Ctrl+Shift+P (או Cmd+Shift+P במק)
  3. הקלד "Screenshot" ובחר "Capture node screenshot"
  4. לחץ על הקונטיינר הכחול למטה
  5. שמור את התמונה בשם "og-image.png" בתיקיה public/images/

או: פשוט צלם מסך של הקונטיינר הכחול ועבד עליו בעורך תמונות
